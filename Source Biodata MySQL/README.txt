Aplikasi Biodata CRUD dengan MySQL

1. Import database `biodata.sql`
2. Pindahkan folder `android` ke dalam folder `xampp/htdocs/`
3. Hubungkan wifi laptop dan device kamu dengan Hotspot/Jaringan yang sama
4. Ubah IP Address pada `Server.java` menjadi IP laptop kamu.
5. Jangan lupa bantu share ke temen-temen kamu yaaa
